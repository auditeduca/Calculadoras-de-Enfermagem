Recursos Assistivos

Navegação inclusiva e personalizada. Conheça as ferramentas que garantem o acesso universal às nossas calculadoras clínicas e conteúdos de enfermagem.

Nov, 2025 Leitura: 8 min Acessibilidade Web

Desenvolvemos a plataforma Calculadoras de Enfermagem com o compromisso de não deixar ninguém para trás.

01 Barra de Acessibilidade

Localizada no topo de todas as páginas, nossa barra de ferramentas permite ajustes imediatos na interface para melhor conforto visual e legibilidade.

Escala de Texto

Controles (A+) e (A-) para aumentar ou diminuir a fonte em até 200% sem perda de funcionalidade.

Modos de Contraste

Opções de Alto Contraste (preto/branco) e Contraste Invertido para reduzir a fadiga ocular.

Preferências Salvas Suas escolhas de acessibilidade são armazenadas no navegador, garantindo que o site mantenha o seu estilo em todas as visitas.

02 Fonte OpenDyslexic

Oferecemos a opção de ativar a fonte OpenDyslexic, desenvolvida para mitigar erros comuns de leitura causados pela dislexia. As letras possuem bases mais "pesadas", o que ajuda o cérebro a identificar a orientação correta dos caracteres.

03 VLibras - Tradução para Libras

Como utilizar o tradutor virtual:

  1. 1 Clique no ícone de mãos no canto direito da tela.
  2. 2 Aguarde o carregamento do avatar virtual (personagem).
  3. 3 Selecione o texto que deseja traduzir para a Língua Brasileira de Sinais.

04 Compatibilidade com Leitores de Tela

Nossa estrutura HTML5 semântica e o uso de atributos ARIA garantem a melhor experiência possível com softwares de leitura:

Software Ambiente Status
NVDA / JAWS Windows (Desktop) Totalmente Compatível
VoiceOver macOS / iOS (Apple) Totalmente Compatível
TalkBack Android Totalmente Compatível

05 Navegação por Teclado

Atalho Ação Realizada
Tab Avança para o próximo link ou botão.
Shift + Tab Retorna para o elemento anterior.
Enter Ativa o link, botão ou abre menu.
Esc Fecha janelas modais ou menus suspensos.
