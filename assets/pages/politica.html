Política de Privacidade

Proteção de dados e privacidade dos usuários

Jan, 2025 Leitura: 10 min

Respeitamos sua privacidade e protegemos seus dados pessoais em conformidade com a Lei Geral de Proteção de Dados (LGPD).

01 Coleta de Dados

Coletamos apenas dados necessários para fornecer nossos serviços:

  • Informações de navegação: Como você usa o site
  • Preferências: Acessibilidade e idioma
  • Dados de uso: Anônimos e agregados
  • Contato: Opcional para suporte

02 Uso de Dados

Seus dados são usados exclusivamente para:

  • Melhorar nossos serviços
  • Personalizar sua experiência
  • Análise de uso e performance
  • Comunicações (com consentimento)
  • Conformidade legal

03 Seus Direitos (LGPD)

Você tem direito a:

  • Acesso: Conhecer quais dados temos
  • Correção: Atualizar informações incorretas
  • Exclusão: Solicitar remoção de dados
  • Portabilidade: Receber dados em formato aberto
  • Revogação: Retirar consentimento

04 Segurança de Dados

Implementamos medidas de segurança robustas:

  • Criptografia SSL/TLS
  • Proteção contra acessos não autorizados
  • Auditoria de segurança regular
  • Conformidade com padrões internacionais
  • Backup e recuperação

05 Compartilhamento de Dados

Seus dados não são compartilhados com terceiros sem consentimento, exceto:

  • Requisitos legais
  • Proteção de direitos
  • Segurança pública

06 Retenção de Dados

Mantemos dados apenas pelo tempo necessário:

  • Dados de navegação: 90 dias
  • Preferências de usuário: Enquanto ativo
  • Dados de contato: Conforme solicitado
  • Logs de segurança: 1 ano

07 Contato e Reclamações

Para questões de privacidade ou reclamações:

  • E-mail: ciadeenfermagem@gmail.com
  • Tempo de resposta: 10 dias úteis
  • Escalação: ANPD (Autoridade Nacional de Proteção de Dados)
